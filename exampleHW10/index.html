Visualization

One cool thing to show would be how the number of corgis born over time is different between the different countries.

Select Country:

Note, I used some of the vega-lite axis documentation to change the font size of the year labels and make a line mark with points. I also decided to use the default “nominal” values for the x-axis instead of “temporal” since I didn’t want to bother with re-formatting the data.

I chose the line plot to show trends, but wanted to add in the circles to make sure the viewer understood these are discrete measurements.
